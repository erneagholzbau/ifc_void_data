# ifc_void_data

BlenderBIM Addon: aggregates void data from IFC models from various disciplines: <br>
* voids (AU) (required)
* architecture (A) (optional)
* engineering (optional)
    * fire rating (BR)
    * electrical (E)
    * heating (H)
    * heating/cooling ceilings (HKD)
    * ventilation (L)
    * cooling (K)
    * sanitary (S)
    * sprinklers (SP)

## requires
* blender 2.92
* blenderbim 0.0.201207
